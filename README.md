# Moderation Hitbox Demo — Fabric 1.21.11

Безопасный демонстрационный мод для примера модерации.

## Что делает

- `=` увеличивает визуальный demo-box вокруг сущностей.
- `-` уменьшает визуальный demo-box.
- Значение показывается над хотбаром.
- Мод меняет только `EntityRenderer#getBoundingBox`, то есть визуальную сторону рендера.

## Что НЕ делает

- Не меняет реальные hitbox/dimensions сущностей.
- Не меняет reach, attack range, collision, packets, combat checks.
- Не помогает попадать по игрокам/мобам на сервере.

## Как собрать

Нужны:

- Java 21
- Gradle 8/9 или IntelliJ IDEA с Gradle
- Интернет для загрузки зависимостей из Fabric Maven

Команда:

```bash
gradle build
```

Готовый `.jar` появится в:

```text
build/libs/
```

## Как проверить

1. Установи Fabric Loader для Minecraft `1.21.11`.
2. Положи собранный `.jar` в папку `mods`.
3. В игре нажми `F3+B`, чтобы включить показ ванильных hitbox.
4. Нажимай `=` и `-`, чтобы менять визуальный demo-box.

## Примечание

Если цель — обучение модераторов, можно записать два коротких клипа: обычный `F3+B` и затем визуально увеличенный demo-box, объяснив, что мод не меняет реальные игровые проверки.
